1. Chức năng thêm, sửa, xóa mô phỏng sử dụng fake REST API do đó cần phải cài đặt JSON server để view data, lưu data & sửa data.
	Link cài đặt JSON server: https://github.com/typicode/json-server

2. Xem hướng dẫn cài đặt JSON server tại đây: https://youtu.be/5UJGRHNZc9M?t=212

3. Sau khi cài đặt JSON server, khởi động json server trên terminal bằng dòng lệnh:
	json-server --watch db.json

4. Load lại trang web để xem các dữ liệu được tải lên.
